Tobacco Usage 1995-2010

This data was taken from the Center for Disease Control. The data consists of survey results regarding the usage of tobacco by adults in the United States.

The data is organized by State/Territory, with records present between the years 1995 and 2010. The data consists of percentage responses broken down into the following categories: 'Smoke everyday', 'Smoke some days', 'Former smoker' and 'Never Smoked'. This data provides a useful insight into the trends regarding tobacco use over the past two decades. Additionally, it allows the user to examine the relative success of policies in reducing tobacco use in each State.

This webpage allows users to view all of the data, or only the data for a particular State. Also, users are able to select whether they would like to see the most recent data first, or view it in chronological order.
